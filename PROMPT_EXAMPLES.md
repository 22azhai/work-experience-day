# Prompt Examples for Students

Use these prompts with Copilot. Keep changes small and visible.

## Design prompts
- "Change the background gradient to sunset colors."
- "Make the title text bigger and bolder."
- "Use a different font for headings only."
- "Round the corners of the main card more."
- "Add a subtle shadow to the button."
- "Increase spacing between sections so the page is easier to read."

## Content prompts
- "Rewrite the headline to sound more exciting for teenagers."
- "Add a short subtitle that explains what Alexa does."
- "Add a call-to-action sentence above the button."
- "Change the fun facts text to be more playful."
- "Add one more bullet point to the intro list."

## Interaction prompts
- "Make the button color change when I hover over it."
- "Animate the button slightly when clicked."
- "When the facts open, fade them in smoothly."
- "Change the button text to include an emoji-free icon style like [>] and [v]."

## Image prompts
- "Make the Alexa image in the top-right corner slightly larger."
- "Move the Alexa image lower so it does not overlap content."
- "Add a second image below the card using the images folder."

## Accessibility prompts
- "Improve color contrast so text is easier to read."
- "Check that buttons and images have clear labels and alt text."
- "Make sure the page still looks good on mobile screens."

## Safe prompt pattern
Use this format for best results:

"Edit only `index.html`. Make one small change: <your change>. Keep existing layout and deployment setup."
